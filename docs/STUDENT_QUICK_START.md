## üöÄ Super Quick Start (TL;DR)

If you just want to get started quickly:

```bash
# 1. Clone or use the template
git clone https://github.com/dholtz/ai-upskilling-template.git my-project
cd my-project

# 2. Customize your project
cd app
./setup_new_project.sh my-project-name

# 3. Set up and run
make install
make run
```

Then visit: http://localhost:5000
